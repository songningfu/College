# College

《College》是一款以关系与成长为主轴、带回望意味的大学人生模拟游戏。

玩家会重新经历一段大学青春：在每天有限的时间与精力里安排行动、结识舍友与同学、推进关系、触发事件，并在一次次看似普通的日常里，慢慢走向那些真正会留下记忆的人和时刻。

## 游戏简介

这不是一款只讲恋爱、也不是一款只讲数值经营的校园游戏。`College` 想做的是另一种东西：它借由卡牌式日程安排、事件演绎、轻检定和关系推进，把“大学生活”做成一段既能玩、又能回味的经历。

它关心的不是单纯的升级或攻略效率，而是那些更具体的感受：第一次来到大城市的兴奋、面对更广阔世界时的拘谨、和舍友慢慢熟起来的过程、军训后的疲惫、社团与班级里逐渐长出来的关系，以及某段关系终于跨过日常、开始变得特别的那个瞬间。

## 当前 Demo 范围

当前版本聚焦在**开学前几周**，目标是做出一个高完成度的 Demo 样片，而不是提前铺开完整大学四年内容。

当前 Demo 的体验主线大致为：

- 入学与报到
- 舍友初识与宿舍生活开场
- 军训与适应期
- 班级 / 社团 / 手机关系推进
- 第一次约会收束

## 当前开发状态

当前工程已经完成了核心循环系统，并可通过内置测试场景验证 21 天 Demo 规则，包括：

- 三时段日程推进
- 属性 / 资源 / 关系系统
- 发牌式行动卡循环
- 固定事件、条件事件与随机事件
- 军训与课程表机制

同时，项目内已经搭起多层 UI 骨架，但 UI 主链仍在继续整合中。现阶段的重点是把**核心循环、UI 表现和剧情内容**真正接成一条可演示、可感受的体验链。

## 核心特色

- **大学日常 + 关系成长**：不是纯攻略，而是通过日常安排慢慢养出关系变化
- **发牌式行动选择**：每个时段从手牌里做取舍，形成节奏与差异
- **事件演绎层**：对白、选项、轻检定与结算共同承担“经历感”
- **回望式气质**：既有青春感，也有对遗憾、错过和自我理解的回看
